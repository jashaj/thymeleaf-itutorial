This exercise introduces concepts from chapter 9 of the Thymeleaf + Spring tutorial.

  1. Two beans, releaseDate of type java.sql.TimeStamp and price of type have been set into the context.
  2. Formatters for these types have also been registered using the Spring Conversion Service.
  3. Use the new ${{..}} syntax to format these beans using the Conversion Service.
  4. You can check the proposed solution by pressing the Show solution button () above the code editor.
